Project name

Aurora Bookstore

  1. Short project description (Business needs and system features)

For people who love books, here comes the new bookstore where everyone can find what they’re looking for. From novels to poetry, from science to science-popular books and magazines.

The main user roles (actors in UML) are:

•        Anonymous User – can only view the information pages and try few sample tests without saving test results.

•        Registered User – can have profile information, update favorite books, etc

•        Administrator (extends Registered User) – can manage (create, edit user data and delete) all Registered Users, as well as Tests and Test Results.
